# Changelog

Все значимые изменения в проекте MeetScribe документируются в этом файле.

## [Unreleased]

### Добавлено
- Структура проекта: виртуальное окружение, зависимости, директории (`src/`, `tests/`)
- Модуль конфигурации (`src/config.py`): загрузка/сохранение настроек, переопределение через переменные окружения
- База данных SQLite + FTS5 (`src/storage/database.py`): CRUD-операции, полнотекстовый поиск по встречам
- Markdown-экспортер (`src/storage/exporter.py`): экспорт протоколов встреч в `.md` файлы
- Аудиорекордер (`src/audio/recorder.py`): захват микрофона и системного звука (WASAPI loopback), двухпоточная запись
- Аудиомиксер (`src/audio/mixer.py`): объединение микрофонного и системного аудио в один WAV-файл
- Транскрайбер (`src/ai/transcriber.py`): транскрибация аудио через OpenRouter/Gemini API с разбивкой на чанки и повторными попытками
- AI-саммарайзер (`src/ai/summarizer.py`): генерация структурированного протокола встречи с настраиваемым промптом
- Оболочка приложения (`src/app.py`, `src/main.py`): главное окно CustomTkinter с боковой панелью навигации (Запись, История, Настройки)
- Экран записи (`src/ui/recording_view.py`): таймер, индикаторы уровня звука (VU-метры), полный пайплайн записи → транскрибация → саммари
- Просмотр транскрипта/саммари (`src/ui/transcript_view.py`): вкладки, редактирование названия, копирование, экспорт в .md
- История встреч (`src/ui/history_view.py`): список карточек с поиском FTS5, навигация к деталям
- Панель настроек (`src/ui/settings_view.py`): API ключ, выбор модели, тема, аудиоустройства, горячие клавиши, шаблон промпта
- Глобальные горячие клавиши (`src/ui/hotkeys.py`): Ctrl+Shift+R для старта/остановки записи (pynput)
- Сборка PyInstaller в один `.exe` файл
